Como funciona a fidelização

O programa de fidelização transforma pedidos finalizados em pontos para o cliente. Esses pontos ajudam a identificar clientes recorrentes e benefícios disponíveis.

Pontuação

A cada R$ 1,00 em pedido finalizado, o cliente ganha 1 ponto.

Bônus

Pedidos a partir de R$ 150,00 recebem 20 pontos extras automaticamente.

Benefício

A cada 100 pontos acumulados, o sistema gera 1 benefício para o cliente.

Níveis

  • Bronze: até 99 pontos acumulados.
  • Prata: a partir de 100 pontos acumulados.
  • Ouro: a partir de 300 pontos acumulados.

Sem duplicidade

Um mesmo pedido só gera pontos uma vez. O histórico registra o pedido para evitar pontuação duplicada.

Relatório

A tela de fidelização mostra pontos atuais, benefícios, nível do cliente e histórico de pontos gerados.

Fluxo no sistema

1. Pedido O usuário cadastra ou atualiza um pedido no BakeFlow.
2. Status Quando o status fica Finalizado, o evento é enviado.
3. Microserviço O serviço de fidelização calcula pontos e benefícios.
4. Consulta O resultado aparece no relatório de fidelização.
